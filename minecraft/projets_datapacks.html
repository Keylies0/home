Coordonnées dans la barre d'action

Une fois ce datapack activé dans le monde, chaque joueur voit ses coordonnées XYZ affichées dans sa barre d'action (actionbar) en temps réel.
Les coordonnées sont arrondies à l'entier.

Télécharger ce datapack
Une version one command est aussi disponible.

Localisation de la mort

Une fois ce datapack activé dans le monde, tout joueur qui meurt reçoit un message dans son propre tchat décrivant la localisation de sa mort.
La description comprend les coordonnées XYZ et la dimension.
Les coordonnées sont arrondies à l'entier.
Les dimensions par défaut sont notées : Surface (Overworld) ; Nether ; End. Toute autre dimension est notée explicitement comme ci-contre mod_ou_datapack:dimension.

Télécharger ce datapack
Une version one command est aussi disponible.
